# life-list  
This repo based [greyli-tutorial](https://tutorial.helloflask.com).  
## How to use it  

### create me.html  
1. edit markdown file, aboutMe.md  
2. run md2html.py to create me.html

### flask steps  
1. install python3  
2. `pip install -r requirements.txt`  
3. `flask run`
